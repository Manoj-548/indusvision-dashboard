# IndusVision Dashboard Overview

## Features
- Source file inventory (7768 files)
- Knowledge extraction to KnowledgeEntry
- RAG agent with LlamaIndex + Chroma + Ollama
- Responsive dashboard charts
- Annotation API stubs

## Usage
1. python consolidate_all.py
2. /dashboard/agent/ chat with knowledge

Built with Django, Celery, LlamaIndex, LangChain.
